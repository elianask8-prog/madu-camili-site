---Madu Lash Studio & Camili Liz

Site institucional desenvolvido para o salão Madu Lash Studio & Camili Liz.

---Sobre o projeto

O objetivo deste projeto foi criar um site moderno, responsivo e otimizado para apresentar os serviços oferecidos pelo salão, facilitando o contato com clientes através do WhatsApp e Instagram.

---Funcionalidades

- Página inicial com banner
- Seção da equipe
- Galeria de trabalhos
- Lightbox para ampliar imagens
- Botão flutuante do WhatsApp
- Botão voltar ao topo
- Menu responsivo
- Animações ao rolar a página
- SEO básico
- Compatível com dispositivos móveis

---Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub Pages
